# SCMS-Supply-Chain-Delivery-Dashboard

📦 SCMS Supply Chain Delivery Dashboard | Power BI

📖 Project Overview:  
The SCMS Supply Chain Delivery Dashboard is an interactive Power BI project designed to analyze global supply chain shipment data. The dashboard provides valuable insights into shipment performance, freight costs, vendor performance, product groups, and country-wise deliveries.
This project demonstrates the complete data analytics workflow, including data cleaning, data transformation, KPI creation, and interactive dashboard development using Microsoft Power BI.

🎯 Project Objectives:  
Analyze overall shipment performance.  
Monitor total freight costs and shipment value.  
Identify top-performing vendors.  
Compare shipment modes and freight costs.  
Analyze product group performance.  
Evaluate country-wise shipment distribution.  
Build an interactive dashboard for business decision-making.

🧹 Data Cleaning & Preparation:  
The dataset was cleaned using Power Query.

The following data preparation steps were performed:  
Removed duplicate records.  
Verified that no missing values existed.  
Corrected data types.  
Converted date columns into Date format.  
Converted numeric columns into appropriate numeric data types.  
Replaced conversion errors in Freight Cost (USD) and Weight (Kilograms) with 0 to enable accurate calculations and visualizations.  
Validated categorical fields such as Country, Vendor, and Shipment Mode.

📈 Key Business Insights:  
Identified countries with the highest shipment volumes.  
Compared freight costs across different shipment modes.  
Determined the highest-performing vendors based on line item value.  
Evaluated product group contribution to total shipment value.  
Monitored shipment distribution using interactive filters.  
Enabled users to explore data dynamically through slicers.
